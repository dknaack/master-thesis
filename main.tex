  Abstract              = {
    A number is rational if and only if its decimal representation is eventually periodic.
    Similarly, a number is a quadratic irrational if and only if it its continued fraction is eventually periodic.
    This raises a natural question: Is there a representation which is periodic
    if and only if the number is a cubic irrational?
    Although this question was posed by Charles Hermite nearly two
    centuries ago, it remains unanswered to this day.

    The earliest approach was due to Jacobi and Perron,
    who developed a generalization of the Euclidean algorithm to arbitrarily many numbers.
    Their algorithm uses the smallest number and calculates the remainder with respect to the others.
    Since then, many variants have emerged, each selecting different numbers based on a new algorithm.
    However, usually only a single strategy is considered.

    In this thesis, a broad class of multidimensional continued fractions is presented,
    which combines all of these algorithms under a unified framework.
    It is shown that a specific subset of them converges
    and that every periodic fraction leads to an algebraic number,
    but the converse still remains open.
    Nevertheless, an experimental analysis on these fractions
    suggests that many cubic irrationals have a periodic representation
    under certain algorithms.

  Zusammenfassung       = {
    Im Jahr 1839 stellte Charles Hermite die Frage, ob jede reelle Zahl durch eine
    Folge ganzer Zahlen dargestellt werden kann, die genau dann periodisch ist,
    wenn die dargestellte Zahl eine kubisch irrationale Zahl ist.
    Während Kettenbrüche dieses Problem für quadratische Irrationale lösen,
    ist bis heute keine analoge Darstellung für algebraische Zahlen höheren Grades bekannt.
    Diese Arbeit untersucht einen neuen Zugang zu Hermites Frage durch die Konstruktion
    multidimensionaler Kettenbrüche (MDCF), die gewöhnliche Kettenbrüche mithilfe
    einer Verallgemeinerung des euklidischen Algorithmus auf höhere Dimensionen erweitern.

    Es wird gezeigt, dass die meisten multidimensionalen Kettenbrüche konvergieren
    und dass jede periodische Darstellung eine algebraische Zahl beschreiben muss,
    deren Grad nach oben beschränkt ist.
    Diese Ergebnisse beantworten eine Richtung von Hermites Frage in höheren Dimensionen.
    Eine experimentelle Untersuchung ergänzt die theoretischen Resultate und liefert
    numerische Hinweise darauf, dass eine breite Klasse algebraischer Zahlen --
    einschließlich vieler kubischer Wurzeln -- zu periodischen MDCFs führt.
