  Titel auf Englisch    = {On Elementary Properties of a Multi-Dimensional Generalization of the Euclidean Algorithm},
  Autor                 = {Daniel Knaack},

  Abstract              = {
    A number is rational if and only if its decimal representation is eventually periodic.
    Similarly, a number is a quadratic irrational if and only if it its continued fraction is eventually periodic.
    This raises a natural question: Is there a representation which is periodic
    if and only if the number is a cubic irrational?
    Although this question was posed by Charles Hermite nearly two
    centuries ago, it remains unanswered to this day.

    The earliest approach was due to Jacobi and Perron,
    who developed a generalization of the Euclidean algorithm to arbitrarily many numbers.
    Their algorithm uses the smallest number and calculates the remainder with respect to the others.
    Since then, many variants have emerged, each selecting different numbers based on a new algorithm.
    However, usually only a single strategy is considered.

    In this thesis, a broad class of multidimensional continued fractions is presented,
    which combines all of these algorithms under a unified framework.
    It is shown that a specific subset of them converges
    and that every periodic fraction leads to an algebraic number.
    Although the converse still remains open, an experimental analysis on these
    fractions suggests that many cubic irrationals have a periodic
    representation under certain algorithms.
  },
  Zusammenfassung       = {
    Eine Zahl ist rational genau dann, wenn die Dezimaldarstellung periodisch ist.
    Ebenso ist eine Zahl quadratisch irrational genau dann, wenn der Kettenbruch periodisch ist.
    Dies wirft eine natürliche Frage auf:
    Gibt es eine Repräsentation, die genau dann periodisch ist, wenn die Zahl kubisch irrational ist?
    Obwohl diese Frage bereits vor fast zwei Jahrhunderten von Charles Hermite gestellt wurde,
    ist sie bis heute offen.

    Der erste Ansatz kommt von Jacobi und Perron,
    die eine Generalisierung des euklidischen Algorithmus für mehrere Zahlen entwickelt haben.
    Ihr Algorithmus nimmt dabei die kleinste Zahl und berechnet den Rest in Bezug auf den anderen.
    Seitdem sind viele Varianten des Algorithmus entstanden,
    die jeweils unterschiedliche Zahlen auf Grundlage neuer Algorithmen auswählen.
    Jedoch wird in der Regel nur eine einzelne Strategie betrachtet.

    In dieser Arbeit wird eine breite Klasse multidimensionaler Kettenbrüche vorgestellt,
    die all diese Algorithmen in einem einheitlichen Rahmen zusammenfasst.
    Es wird gezeigt, dass eine bestimmte Teilmenge davon konvergiert
    und dass jeder periodische Kettenbruch zu einer algebraischen Zahl führt.
    Auch wenn die Umkehrung weiterhin offen bleibt,
    legt eine experimentelle Analyse dieser Kettenbrüche nahe,
    dass viele kubisch irrationale Zahlen unter bestimmten Algorithmen eine
    periodische Darstellung besitzen.
  },
